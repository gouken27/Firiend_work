## 1. Языки программирования  

### Что такое язык программирования?  
> **Язык программирования** — это способ общения с компьютером.  
> Ты пишешь команды на понятном тебе языке → компьютер их выполняет.

| Это язык программирования? | Почему? |
|----------------------------|--------|
| **Python** | Да — есть синтаксис, интерпретатор/компилятор |
| **HTML** | Нет — это разметка, не выполняет логику |
| **CSS** | Нет — стили, не логика |
| **SQL** | Частично — язык запросов, но не полноценный |

### Почему их так много?  
| Причина | Пример |
|--------|--------|
| **Разные задачи** | Python — универсал, C — системы, JavaScript — веб |
| **Скорость** | C/C++ — быстро, Python — медленнее, но проще |
| **Экосистема** | Python — наука, веб, автоматизация |
| **История** | Старые языки → новые потребности |

## 4. Циклы  

### Зачем? Повторять действия.

| Цикл | Когда использовать |
|------|-------------------|
| `for` | Когда знаешь количество |
| `while` | Пока условие верно |

```python
# for — перебор
for i in range(3):
    print(i)  # 0, 1, 2

# while — пока
count = 0
while count < 3:
    print(count)
    count += 1
```

## 5. Функции  

### Зачем? Не писать одно и то же 100 раз.

```python
def привет(имя):
    return f"Привет, {имя}!"

print(привет("Дима"))  # Привет, Дима!
```

| Часть | Что это |
|------|--------|
| `def` | начало функции |
| `имя` | параметр |
| `return` | что возвращает |

## 6. Файлы  

### Чтение и запись

```python
# Запись
with open("file.txt", "w", encoding="utf-8") as f:
    f.write("Привет, мир!")

# Чтение
with open("file.txt", "r", encoding="utf-8") as f:
    text = f.read()
    print(text)
```

> **Всегда используй `with`** — автоматически закрывает файл.

## 7. Графика на Tkinter  

### Простое окно с кнопкой

```python
import tkinter as tk

root = tk.Tk()
root.title("Моё окно")
root.geometry("300x200")

def клик():
    label.config(text="Нажато!")

btn = tk.Button(root, text="Нажми", command=клик)
btn.pack(pady=20)

label = tk.Label(root, text="Готов")
label.pack()

root.mainloop()
```

| Виджет | Зачем |
|-------|------|
| `Button` | кнопка |
| `Label` | текст |
| `Entry` | поле ввода |
| `Canvas` | рисование |
